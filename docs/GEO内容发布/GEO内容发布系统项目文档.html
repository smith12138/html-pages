1 产品概述

Geo-Admin -- GEO内容发布系统

GEO内容发布系统是一套基于 Gin-Vue-Admin 框架构建的 SEO/GEO 内容自动生成与多平台发布管理平台。系统利用大语言模型（LLM）自动生成 SEO 优化文章，并通过 AdsPower 反指纹浏览器实现多平台自动化发布。

核心定位：针对巴西 iGaming 市场的 SEO 内容自动化生产与分发系统。从选题策划、AI 内容生成、内容审核到多平台自动发布，实现内容营销全流程自动化。

5 模块一：账号环境

AdsPower 集成管理 浏览器自动化

账号环境模块是系统与 AdsPower 反指纹浏览器的集成层，负责管理浏览器配置文件、代理 IP、发布节点等基础设施。每个浏览器环境拥有独立的指纹特征和网络身份，确保多平台操作的隔离性与安全性。

注意：未选择「当前 Publisher」时，环境/分组/代理操作将使用服务端默认 AdsPower 地址。列表加载后会自动探测运行状态，请勿连续快速翻页以免触发 Publisher 限流。

5.1 节点管理

  • 管理 Publisher 发布节点（运行 AdsPower 的服务器实例）
  • 通过端口/URL 连接到节点
  • 节点状态监控与健康检查
  • 支持多节点分布式部署

5.2 分组管理

  • 将浏览器环境按业务需要分组管理
  • 支持按平台、用途、地区等维度分组
  • 批量操作分组内环境

5.3 代理管理

  • 管理浏览器环境使用的代理 IP
  • 支持 HTTP/SOCKS 代理协议
  • 代理可用性检测
  • IP 轮换策略配置

5.4 环境列表

  • AdsPower 浏览器配置文件管理
  • 字段：编号/ID、分组、名称、IP、创建时间、状态、账号平台、标签、备注
  • 环境命名规则：reddit_main_hs_001
  • 操作：新建环境、实时列表、编辑、删除
  • 「绑定 Publisher」-- 绑定环境到发布节点
  • 账号平台：hostget / reddit / tumblr / thinx / quora + 邮箱账号

5.5 发布平台

  • 配置内容发布的目标平台
  • 覆盖 11 个平台：Reddit, Quora, Tumblr, Discord, YouTube, Hostget, Bluspy, Brucky, JackpotBoss, ThinK, Thinlr
  • 平台参数配置与发布规则
6 模块二：内容中心

AI 驱动的内容生产引擎 核心模块

内容中心是系统的核心模块，涵盖从选题策划到内容生成的完整流程。通过 LLM 大语言模型，将中文种子词自动扩展为英文 SEO 优化文章，支持图文和纯文本两种格式。

6.1 选题库

当前共 72 个选题。

  • 字段：ID、种子词、关键词、Plan详情（JSON）、生成次数、状态、创建时间、任务ID
  • 种子词示例：「巴西最好的iGaming平台」「巴西最好的slots游戏平台」「巴西博彩游戏」
  • Plan详情：JSON 配置，包含 source、core_keyword 等参数
  • 操作：新建选题、AI 生成选题、生成内容、编辑、删除
  • 筛选：关键词搜索、状态过滤（全部/就绪/生成中等）

6.2 内容库

当前共 2,162 篇文章（217 页）。

  • 字段：ID、关键词、标题、初稿（HTML）、类型、状态、发布平台、写作简略、时间
  • 内容类型：图文 纯文本
  • 文章语言：英文，SEO 优化
  • 标题示例："Why Is Youkonggang Good for Low-Spec PCs..."
  • 操作：批量发布、追加文章、批量删除、编辑、预览

6.3 生成历史

  • 追踪内容生成任务的执行历史
  • 记录每次生成的参数、结果、耗时
  • 便于回溯和优化生成策略

13 核心技术概念

AdsPower 反指纹浏览器

用于管理多个浏览器配置文件的反检测浏览器。每个配置文件拥有独立的浏览器指纹（Canvas、WebGL、UA 等），模拟真实用户身份，避免多账号被平台关联检测。

Publisher 发布节点

运行 AdsPower 的服务器实例，作为浏览器自动化的执行节点。系统通过 API 远程控制节点上的浏览器环境执行发布任务。

Environment 浏览器环境

单个浏览器配置文件，包含唯一指纹、代理 IP 和平台账号绑定。命名格式如 reddit_main_hs_001，表示平台_用途_标识_序号。

GEO / SEO 优化

SEO（搜索引擎优化）：针对传统搜索引擎的内容优化。GEO（生成式引擎优化）：针对 AI 生成式搜索引擎（如 Perplexity、SGE）的内容优化。系统同时覆盖两种优化策略。

种子词与关键词扩展

种子词：用户输入的基础关键词（中文），如「巴西最好的iGaming平台」。系统通过 LLM 将种子词扩展为多个相关关键词，并生成对应的英文 SEO 文章。

Plan 详情与 Prompt 模板

Plan详情：JSON 格式的内容生成配置，包含 source、core_keyword 等参数。Prompt模板：针对不同平台和文章风格的 LLM 提示词模板，控制生成内容的质量和方向。

内容类型：图文 vs 纯文本

图文 包含图片和文本的富内容，适合博客平台和论坛。纯文本 仅包含文字内容，适合问答社区和评论区。不同平台根据其特性使用不同内容类型。

平台动作类型

顶名 点赞/顶帖提升曝光。发布 发布原创内容。外写 外部投稿写作。转发 转发分享已有内容。不同动作对应不同的浏览器自动化脚本。
