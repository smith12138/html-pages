一、产品概述与定位

EngageLab LiveDesk — 极光(Aurora Mobile)旗下全渠道智能客服平台

1.1 产品简介

EngageLab LiveDesk 是极光（Aurora Mobile）推出的全渠道智能客服平台，提供 AI Agent + Human Agent 混合协同模式，支持 Web LiveChat Widget 等多种渠道接入。

服务模式AI Agent 自动应答 + Human Agent 人工处理

十、核心业务流程

对话全生命周期管理与分配机制

10.1 对话生命周期

用户发起对话
→
系统创建对话
→
AI自动响应 / 手动分配
→
客服处理对话
→
对话已解决

10.2 对话状态流转

当前状态可转换为触发条件
OpenResolved / On Hold / Closed客服操作、自动规则
On HoldOpen / Closed定时恢复、手动操作
ResolvedOpen客户发送新消息时自动恢复
Closed—终态，不可恢复
搁置（On Hold）状态是 LiveDesk 的特色设计，允许客服暂挂对话并设置自动恢复时间，避免对话被遗忘。

10.3 对话分配机制

分配方式说明
自动分配系统根据规则自动分配给可用客服
手动分配管理员/客服在详情面板中手动指定处理人
主动认领客服从"未分配"队列中主动认领对话
转交当前客服将对话转交给其他客服或团队
右键分配通过右键菜单快速分配客服或团队
