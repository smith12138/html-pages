8 数据模型与运维

数据库表结构、Redis 键模式、文件命名规范、API 路由、前端页面

PostgreSQL 数据表

表名 用途 关键列
cluster_result_index 所有聚类运行的索引 platform, product, run_id, n_clusters, as_of, status
cluster_risk_alert 高危聚类告警 cluster_id, risk_score, risk_level, factors_json
api_users API 认证 username, hashed_password, role, is_active

Redis 键模式

模式 类型 描述
fg:v1:evt:{platform}:{shard_id} String (MessagePack) 事件数据分片
fg:v1:evtuser:{platform}:{user_id} String (MessagePack) 用户级事件缓存
fg:v1:wm:{platform} String 每个平台的水位线时间戳

输出文件命名

在线 features_{platform}_{product}_{as_of_iso}.parquet
离线 offline_{platform}_{product}_{window_start}_{window_end}.parquet
解释 user_paths/{platform}_{product}_{run_id}.csv
解释 sankey/{platform}_{product}_{run_id}.csv
模型 gang_discovery/{platform}_{product}/{run_id}/
